## Ⅰ. 시장 단가 요약 (톤당 · 가단가)

현장 장비(354.35kW 등) **파쇄만** 하는 비용과, **성상별 이동·위탁 처리** 비용은 **별도 항목**으로 구성합니다.

| 구분 | 항목 | 평균 단가 범위 (톤당) | 비고 |
|---|---|---:|---|
| 현장 파쇄 | 가지·잡목류 파쇄 | 25,000 ~ 40,000원 | 현장 내 깔기·유용(살포 등) 기준 |
| 현장 파쇄 | 뿌리류(근치) 파쇄 | 35,000 ~ 55,000원 | 토사 혼입·장비 마모로 단가 상승 |
| 위탁 처리 | 임목폐기물 성상별 처리 | 70,000 ~ 95,000원 | 순수 폐목재 재활용·매립/소각 위탁 |
| 운반비 | 덤프트럭(25톤) | 15,000 ~ 25,000원 | 이동 거리(20km 내외)에 따라 변동 |

**실무 팁** — 대규모 산지·토목 현장에서 **현장 파쇄 후 톱밥·칩 형태로 전량 유용**(부지 내 살포·복토 등)하면 **위탁 처리비·장거리 운반비**가 빠져 **총액이 크게 낮아질** 수 있습니다. 반대로 **전량 위탁** 시 파쇄+운반+처리를 **중복 계상**하지 않도록 내역 구조를 맞춥니다.

## Ⅱ. 품셈·설계내역서 기준 단가 구성

관급·정식 설계서에서는 **단일 톤(또는 ㎥) 단가**가 아니라, 아래가 **시간당 생산량 Q**로 나뉘어 합산됩니다.

**단가(원/㎥ 또는 원/ton) ≒ (장비손료 + 주연료비 + 노무비) ÷ Q**

| 구성요소 | 내용 |
|---|---|
| 장비손료 | 이동식 임목파쇄기(354.35kW) + **투입용 굴착기**(0.7㎥급·집게 어태치먼트) |
| 주연료비 | 354.35kW(약 480HP) 엔진 **시간당 경유 소모량 × 유가** |
| 노무비 | 장비 조종원 + 보통인부 1~2명(정리·안전) |
| Q | 잡목·가지 **6.0~8.0 ㎥/hr**, 뿌리류 **4.0~5.0 ㎥/hr** (품셈·실적 범위) |

조달청 자원분류(건설기계)에는 **이동식 임목파쇄기 354.35kW**(코드 `0000720504750000`)가 등재되어 있으나, **표준시장단가 DB 일위대가**에는 본건과 1:1 매칭되는 **「임목파쇄」 일위항목이 없어** 현재 `01_…토목_표준단가산출`에서 **미매칭** 상태입니다.

## Ⅲ. 단가 산출 단계별 체크리스트

**1. 장비 조합**

- [ ] 파쇄기 본체(354.35kW) 외 **굴착기 0.7㎥급 + 집게 어태치먼트** 손료가 함께 계상되었는가?
- [ ] 임목 **운반 83m** 구간이 덤프(톤당)인지, 로더·크레인(시간당)인지 **작업 방식과 단위**가 일치하는가?

**2. 생산량 Q**

- [ ] 가지·잡목 **Q = 6.0~8.0 ㎥/hr** 적용 여부 확인
- [ ] 뿌리류 **Q = 4.0~5.0 ㎥/hr** 적용·단가 상승 반영
- [ ] 본건 뿌리류 **11 ㎥** — Q 하향 시 **고정비 분摊**으로 ㎥당 단가가 더 올라감

**3. 유류·손료**

- [ ] 354.35kW 엔진 **시간당 연료소모 × 적용 유가(경유)** 반영
- [ ] 장비 **가동률·대기시간** (이동·세팅·고장) 포함 여부

**4. 노무·안전**

- [ ] 조종원 + **보통인부 1~2명** 포함
- [ ] **안전관리·분진·소음** 대책비가 별도 항목인지 일위대가에 포함인지 구분

**5. 처리 방식(시나리오)**

- [ ] **현장 유용** — 위탁 처리·장거리 덤프 **제외**했는가?
- [ ] **위탁 처리** — `07_…건설폐기물처리` 협회단가(G00000004 운반 등)와 **목재 전용 처리비** 중복·누락 없는가?
- [ ] PE천막·적치장(본건 2). 항목)은 **파쇄 단가에 합산**하지 않았는가?

**6. 단위·환산**

- [ ] 내역 **㎥**, 견적 **톤** 혼용 시 **환산계수(ton/㎥)를 서면에 명시**했는가?
- [ ] 표준단가산출·총괄표 합계에 **수동 입력 단가** 반영 경로를 정했는가?

## Ⅳ. 시장가 vs 품셈 설계단가 — 비교 요약

| 구분 | 시장 가단가 | 품셈·설계단가 |
|---|---|---|
| 성격 | 지역·업체 견적, 작업 방식 반영 | 장비손료+유류+노무 ÷ Q |
| 유리한 경우 | 긴급·소량, 협상 단가 확인 | 관급·정식 예가, 장비 스펙 명확 |
