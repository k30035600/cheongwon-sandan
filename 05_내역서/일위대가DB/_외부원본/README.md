# 일위대가DB — 화성 청원지구

워크스페이스 내 공내역서에서 **추출 가능한 모든 일위대가(품셈 기반)** 를 통합 저장한 폴더.

## 구성 파일

- `일위대가_통합DB.xlsx` — 전 원천 통합(시트별) + `_통합마스터` 시트
- `_master_일위대가.csv` — 통합 마스터 (196건). 컬럼: 출처, 시트, 호표, 코드, 품명, 규격, 단위, 재료비단가, 노무비단가, 경비단가, 합계단가, 비고
- `전기(02)_일위대가_호표.csv` — 66건
- `전기(02)_일위대가_상세.csv` — 410건
- `전기(02)_합산자재.csv` — 128건
- `전기(02)_단가조사.csv` — 129건
- `폐기물(07)_경비_협회단가.csv` — 2건

## 원천 및 범위

| 원천 | 내용 | 비고 |
|---|---|---|
| **전기(02)** | 일대목차(호표 일위대가 단가표)·일위대가 상세(구성)·합산자재·단가조사 | 품셈 기반, 재/노/경 단가 |
| **폐기물(07)** | 경비(26. 협회단가) 2건 | 폐기물 운반·처리 Ton당 |

> **토목 01·04·05·06** 은 원본 XLS에 일위대가·품셈 시트가 **없어 추출 불가**.

## 외부 표준일위대가 통합 (토목)

외부에서 받은 **조달청 표준일위대가 XLSX**(예: `2026년 토목공사 표준일위대가.xlsx`)를
아래 폴더에 넣고 스크립트를 다시 실행하면 **자동으로 헤더를 인식해 통합**된다.

- **투입 위치**: `일위대가DB/_외부원본/` (파일명에 `일위대가` 포함된 `.xlsx`)
- **실행**: `python -X utf8 tools/build_ildae_db.py`
- 헤더에 `품명/공종명 · 규격 · 단위 · 재료비 · 노무비 · 경비 · 합계` 가 있으면
  레이아웃이 달라도 컬럼을 자동 매핑한다.

> ⚠️ 조달청 NPCCS·민간 자료실·공공데이터포털 모두 **로그인/공동인증서**가 필요해
> 에이전트가 자동 다운로드할 수 없다. 사용자가 직접 받아 위 폴더에 넣어야 한다.

## 단가 구조

- **일대목차**: 호표(제 N호)별 1단위 재료비·노무비·경비·합계 단가 → 내역서 수량 × 단가.
- **일위대가 상세**: 각 호표를 구성하는 자원(노무·자재·중기)별 공량·단가(품셈 산출근거).
- **합산자재 / 단가조사**: 자재·중기 단가의 출처(거래가격·물가정보·물가자료).

## 활용

- 내역서 항목 ↔ 일위대가는 **번호·호표 코드**로 정확 매칭(전기), 또는 **품명+규격**으로 매칭.
- 재생성: `python -X utf8 tools/build_ildae_db.py`
